# Peer Review

Reviews your manuscript in three passes:

1. **Technical Review** -- Methods, statistics, validity, reproducibility
2. **Editorial Review** -- Structure, argumentation, clarity, completeness
3. **Synthesis** -- Combines findings into a structured report with severity ratings

The review takes 2-5 minutes depending on document length.

## What you get

A structured report with:
- General assessment
- Identified strengths
- Issues grouped by severity (Critical / Major / Minor)
- Specific recommendations with quoted passages

## Tips

- Markdown and LaTeX files work best. Plain text is also supported.
- Use the "focus" field to direct attention to specific areas (e.g., "statistical methods" or "discussion section").
- The review covers methodology and writing quality. It does not verify factual claims or run statistical tests.
- For best results, ensure the document includes standard sections (abstract, introduction, methods, results, discussion).

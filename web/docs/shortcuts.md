## Navigation

| Action | Shortcut |
|--------|----------|
| Quick open (files, content, references) | <kbd>Cmd/Ctrl</kbd>+<kbd>P</kbd> |
| Open settings | <kbd>Cmd/Ctrl</kbd>+<kbd>,</kbd> |
| Toggle left sidebar | <kbd>Cmd/Ctrl</kbd>+<kbd>B</kbd> |
| Toggle right sidebar | <kbd>Cmd/Ctrl</kbd>+<kbd>J</kbd> |
| Filter file tree | <kbd>Cmd/Ctrl</kbd>+<kbd>F</kbd> (sidebar focused) |

## Files & tabs

| Action | Shortcut |
|--------|----------|
| Save and commit | <kbd>Cmd/Ctrl</kbd>+<kbd>S</kbd> |
| New (same type as current) | <kbd>Cmd/Ctrl</kbd>+<kbd>N</kbd> |
| New tab (Start page) | <kbd>Cmd/Ctrl</kbd>+<kbd>T</kbd> |
| Close tab | <kbd>Cmd/Ctrl</kbd>+<kbd>W</kbd> |
| Switch tab left | <kbd>Cmd/Ctrl</kbd>+<kbd>Option/Alt</kbd>+<kbd>Left</kbd> |
| Switch tab right | <kbd>Cmd/Ctrl</kbd>+<kbd>Option/Alt</kbd>+<kbd>Right</kbd> |
| Split vertically | <kbd>Cmd/Ctrl</kbd>+<kbd>\\</kbd> |
| Split horizontally | <kbd>Cmd/Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>\\</kbd> |

## Editing

| Action | Shortcut |
|--------|----------|
| Bold | <kbd>Cmd/Ctrl</kbd>+<kbd>B</kbd> (editor focused) |
| Italic | <kbd>Cmd/Ctrl</kbd>+<kbd>I</kbd> |
| Link | <kbd>Cmd/Ctrl</kbd>+<kbd>K</kbd> |
| Inline code | <kbd>Cmd/Ctrl</kbd>+<kbd>E</kbd> |
| Find | <kbd>Cmd/Ctrl</kbd>+<kbd>F</kbd> (editor focused) |
| Find and replace | <kbd>Cmd/Ctrl</kbd>+<kbd>H</kbd> |

## AI features

| Action | Shortcut |
|--------|----------|
| Trigger inline suggestion | `++` (two plus signs quickly) |
| Accept suggestion | <kbd>Tab</kbd>, <kbd>Enter</kbd>, or <kbd>Right</kbd> |
| Cycle suggestions forward | <kbd>Down</kbd> |
| Cycle suggestions back | <kbd>Up</kbd> |
| Dismiss suggestion | <kbd>Esc</kbd>, <kbd>Left</kbd>, or click elsewhere |
| Add comment on selection | <kbd>Cmd/Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>L</kbd> |

## Code execution

| Action | Shortcut |
|--------|----------|
| Run selection or current line | <kbd>Cmd/Ctrl</kbd>+<kbd>Enter</kbd> |
| Run entire file | <kbd>Shift</kbd>+<kbd>Cmd/Ctrl</kbd>+<kbd>Enter</kbd> |

## View

| Action | Shortcut |
|--------|----------|
| Zoom in | <kbd>Cmd/Ctrl</kbd>+<kbd>+</kbd> |
| Zoom out | <kbd>Cmd/Ctrl</kbd>+<kbd>-</kbd> |
| Toggle word wrap | <kbd>Option/Alt</kbd>+<kbd>Z</kbd> |

## File explorer

| Action | Shortcut |
|--------|----------|
| Navigate up / down | <kbd>↑</kbd> / <kbd>↓</kbd> |
| Expand folder | <kbd>→</kbd> |
| Collapse folder / go to parent | <kbd>←</kbd> |
| Open file / toggle folder | <kbd>Space</kbd> |
| Rename | <kbd>Enter</kbd> |
| Jump to first / last item | <kbd>Home</kbd> / <kbd>End</kbd> |
